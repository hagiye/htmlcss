What is the Box Model?

Every element in web design is a rectangular box. The CSS box model describes how these boxes are sized and positioned.

The model consists of: content, padding, border, and margin.

Box Model Components

Content

The actual content of the box, where text and images appear.

Padding

Clears an area around the content. The space inside the box.

Border

A border that goes around the padding and content.

Margin

Clears an area outside the border. The space between boxes.
